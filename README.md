# multi-page website

A responsive multi-page website technology education institution.


## Project Overview

This project is a fully responsive website for ABC Academy, built with HTML5 and CSS3. It includes four main pages showcasing the academy's courses, mission, and contact information.

## Pages

1. **Home** (`index.html`) - Landing page with hero section, features, popular courses, and statistics
2. **About** (`about.html`) - Information about the academy's mission, values, and leadership team
3. **Courses** (`courses.html`) - Complete course catalog with pricing options
4. **Contact** (`contact.html`) - Contact form, location information, and office hours

## Features

- **Responsive Design**: Mobile-first approach with media queries
- **Modern CSS**: Flexbox layout, CSS variables, pseudo-elements
- **Accessibility**: Semantic HTML, proper labeling
- **Clean Navigation**: Fixed header with active state indicators
- **Interactive Elements**: Hover states, form validation
